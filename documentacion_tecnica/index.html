Introduccion

Esta documentacion tecnica presenta una introduccion a HTML y CSS.

Esta orientada a personas que estan empezando en el desarrollo web.

HTML es el lenguaje que define la estructura del contenido web.

CSS se encarga de definir el aspecto visual del contenido HTML.

Ambos lenguajes son esenciales para cualquier desarrollador frontend.

Con HTML creamos etiquetas, y con CSS las estilizamos.

Esta guia aborda conceptos basicos y algunas practicas recomendadas.

Tambien aprenderas sobre la organizacion del codigo y selectores.

Se utiliza codigo de ejemplo para facilitar la comprension.

Ademas, se muestra como hacer diseños adaptables a distintos dispositivos.

<html>...</html> <body>...</body>
HTML_Basico

HTML significa HyperText Markup Language.

Utiliza etiquetas para describir los elementos del contenido.

Algunas etiquetas comunes son: <h1>, <p>, <ul>, <a>.

Las etiquetas pueden tener atributos como class o id.

Tambien podemos anidar etiquetas para construir estructuras mas complejas.

<h1>Titulo principal</h1> <a href="#">Enlace</a>
  • Etiqueta <div>: contenedor generico.
  • Etiqueta <img>: para imagenes.
  • Etiqueta <ul> y <li>: listas.
CSS_Basico

CSS significa Cascading Style Sheets.

Se utiliza para aplicar estilos a elementos HTML.

Podemos definir colores, fuentes, margenes y mas.

Un estilo basico se aplica mediante selectores.

Se puede escribir CSS en un archivo externo.

body { background-color: #f0f0f0; } h1 { font-size: 2rem; }
  • Propiedad color
  • Propiedad margin
  • Propiedad padding
Selectores_y_Especificidad

Los selectores permiten apuntar a elementos HTML.

Existen selectores de tipo, clase, ID y combinadores.

La especificidad define que estilo se aplica si hay conflictos.

Un ID tiene mas prioridad que una clase.

Tambien es posible usar pseudoclases y pseudoelementos.

.boton:hover { background-color: blue; } #cabecera { font-weight: bold; }
  • Selectores simples
  • Selectores compuestos
  • Especificidad: inline > ID > clase
Diseno_Responsivo

El diseño responsivo adapta el contenido a diferentes tamaños de pantalla.

Se logra principalmente mediante media queries.

Tambien se pueden usar unidades relativas como %, vw y vh.

Las propiedades flexbox y grid ayudan a distribuir el contenido.

Es importante mantener la usabilidad en todos los dispositivos.

@media (max-width: 600px) { ... } img { width: 100%; height: auto; }
  • Mobile first
  • Flexbox
  • Grid
